# customer_behavior_analysis
**Overview**

This project presents an end-to-end data analytics workflow focused on extracting actionable insights from a structured dataset. It involves loading and processing data using Python, performing exploratory data analysis (EDA) to understand patterns and trends, and cleaning the dataset to ensure accuracy and consistency.

Further analysis is conducted using PostgreSQL, where SQL queries are used to derive meaningful insights and support data-driven decisions. Finally, the results are visualized through an interactive Power BI dashboard, enabling easy interpretation of key metrics and trends.

The project highlights practical skills in data analysis, SQL, and data visualization, demonstrating the ability to transform raw data into valuable business insights.

**Tools & Technologies Used :**

Python – Data loading, cleaning, and exploratory data analysis (EDA)

Pandas, NumPy, Matplotlib, Seaborn – Data manipulation and visualization

PostgreSQL – Writing and executing SQL queries for deeper analysis

Power BI – Building an interactive dashboard for insights visualization
