# customer_behavior_analysis
Overview

This project demonstrates an end-to-end data analytics workflow, starting from data loading and exploration to building an interactive dashboard. The goal is to extract meaningful insights from raw data and present them in a clear, visual format for decision-making.

Tools & Technologies Used

Python – Data loading, cleaning, and exploratory data analysis (EDA)

Pandas, NumPy, Matplotlib, Seaborn – Data manipulation and visualization

PostgreSQL – Writing and executing SQL queries for deeper analysis

Power BI – Building an interactive dashboard for insights visualization
